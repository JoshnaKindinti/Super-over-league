# Super Over League 🏏

A responsive, visually engaging landing page for a cricket league built using fundamental React concepts (JSX, Elements) and CSS Flexbox layout.

## 🛠️ Features
- Built dynamically using **React 17** (without create-react-app overhead).
- **Fully Responsive Design:** Adapts smoothly from mobile screens up to desktop views.
- Clean layout structures utilizing **CSS Flexbox** and modern CSS resets.

## 💻 Tech Stack
- **Frontend:** React.js (via CDN link), HTML5, CSS3
- **Fonts:** Roboto via Google Fonts
